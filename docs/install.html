Installation

Install the desktop app.

The download page points to the latest GitHub release. The app currently targets desktop platforms: macOS, Windows, and Linux.

macOS

  1. Download the macOS installer from the latest release.
  2. Open the DMG or ZIP and move the app into Applications.
  3. If macOS blocks the app, open System Settings, then Privacy & Security, and choose Open Anyway.

The warning appears because development builds are not signed and notarized yet. Signed releases can be added later without changing this page.

Windows and Linux

Download the installer matching your system from the latest release. The download page looks for Windows assets ending in EXE or MSI, and Linux assets ending in AppImage, DEB, or RPM.

Release checklist

  1. Create local installers for macOS, Windows, and Linux when the app is ready.
  2. Create a GitHub release with a version tag.
  3. Upload installer assets with clear platform names, such as TUE-Study-Hub-macOS.dmg.
  4. Open the download page and confirm the primary button points to the right installer.
